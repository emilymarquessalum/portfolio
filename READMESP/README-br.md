
 # Portfolio

 Criado para mostrar minhas criacoes, me desafiar, e talvez me ajudar a conseguir um emprego no processo!

 ## Sumario das Secoes

 Vamos dar uma olhada rapida no que o portfolio tem!
 
 ### Configuracoes do Sistema

 Essa secao e aberta quando o usuario carrega a pagina.
 Nela, voce pode fazer Configuracoes para customizar e melhorar sua experiencia!
 Atualmente, apenas para selecionar o idioma do portfolio (ingles ou portugues).

 ### Header

Com tema de explorador de arquivos de um IDE, links para diferentes secoes do portfolio.

 
 ### Techs

 Lista as tecnologias que eu conheco.

 ### Projects

Com tema de um array de objetos, mostra e linka para meus principais projetos.
 

 ### Perfil

Com tema de um dictionary, mostra informacoes sobre mim.

 ### Contacts

Tem um formulario para me contatar, lista redes sociais onde podem me encontrar.
